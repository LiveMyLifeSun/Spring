                                Jude 竹
=========================================================================

【概要】

  Jude(Java and UML Developers' Environment) は、Java 言語とUML1.4
  (Unified Modeling Language) によるオブジェクト指向設計をサポート
  する設計支援ツールです。

【機能・特徴】

  今回の竹バージョンでは以下のような基本機能があります。
   ・UML1.4 のクラス図、ユースケース図、ステートチャート図、アクティビティ図、
     シーケンス図、コラボレーション図、配置図、そして、コンポーネント図へ対応
     し、ソフトウェア構造の視覚化をサポート
   ・クラス図では、オブジェクト図、パッケージ図、サブシステム図、ロバストネス図
     もサポート
   ・モデルデータから Java ソースファイルのテンプレートを生成
   ・Java ソースファイルからモデルデータを生成
   ・図を画像形式で保存、またWordなどへの貼り付け

  特に、以下のような特徴があります。
   ・斜め方向などへの自由なスクロール（右ドラッグ）
   ・マウスによるズーム
   ・全体マップの採用で、大きな図内を自由に視点を移動して編集
   ・無限回の UNDO/REDO
   ・パッケージ・ツリー表示、クラス階層・ツリー表示が可能
   ・美しい自動レイアウト機能
   ・100% pure java アプリケーション

  ※新しいバージョンでは、以前のバージョンで作成した図が読めない可能性があります
    (ご了承ください)

【動作環境】

  J2SE1.4.1以降 がインストールされていることが必要です。
  （Windows2000, XPにて動作を確認しています。）

【インストール・起動方法】

  1. J2SEのインストール
    推奨されるJ2SEがインストールされていない場合は、あらかじめインストールします。
    J2SEのインストール方法は、J2SEのホームページを参照ください。

  2. Judeのインストール
    古いバージョンがインストールされている場合は、あらかじめアンインストールしてください。

    ◎インストーラ利用の場合
      ダウンロードしたインストーラ（jude_xxx-setup.exe)を実行します。
      ※javaの特別なオプション設定などが必要な場合は、jude.batを編集してください。
      ※Windows98、MEの場合は、jude.batを編集して、JUDE_HOMEを設定してください

    ◎インストーラを利用しない場合
      ダウンロードしたアーカイブ（jude_xxx.zip)を適当なフォルダに解凍します。

  3. 起動

    ◎起動方法１(インストーラ使用の場合)
    スタートアップメニュー、または、デスクトップのショートカットから起動します。
    また、judeファイルをダブルクリックして、Judeを起動します。

    ◎起動方法２
    Windowsユーザで、J2REがインストールされている場合は、jude_take.jarを
    ダブルクリックするだけで起動します。

    jarのダブルクリックで起動できない場合は、以下の方法で起動します。

    ◎起動方法３
    Windowsユーザは、アーカイブに含まれるバッチファイル(jude.bat)を利用できます。
    (自分の環境にあわせて編集してください)。

    ◎起動方法４
    コマンドラインから以下のように起動します。
    > java -jar jude_take.jar

  ※Export HTML を利用する場合は、JDKが提供するtools.jarを、
    jude_take.jarと同じフォルダにコピーしてください。 

    うまく起動できない場合、jude-info@esm.co.jp にお問合わせ下さい。

【アンインストール】
  インストーラでインストールした場合：
    1. スタートメニューから"Uninstall Jude"を実行してください。

  インストーラを利用していない場合：
    1. jude_take.jar を削除してください。
    2. コピーしたtools.jarを削除してください。
